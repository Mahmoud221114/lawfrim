## 📌 Project Overview
This project is a modern and responsive website designed for a law firm.  
It focuses on delivering a clean user interface and making it easy for clients to explore services and contact the office.

---

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript (Vanilla)

---

## ✨ Features
- ⚖️ Professional and clean design suitable for law firms  
- 📱 Fully responsive on all devices  
- 📂 Well-structured sections (Home, About, Services, Contact)  
- 🎬 Smooth animations for better user experience  
- 💬 WhatsApp button for direct communication  

---

## 📈 Challenges
- Creating a design that reflects trust and professionalism  
- Keeping the UI simple while maintaining a modern look  
- Improving user experience in a service-based website
